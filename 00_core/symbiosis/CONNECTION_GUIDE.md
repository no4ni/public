# 📱 ПОДКЛЮЧЕНИЕ ТЕЛЕФОНА К ПК
# ==============================
# Время: 09:09
# Статус: Веб-сервер запущен на Python 3.6

## 🌐 АДРЕС ДЛЯ ПОДКЛЮЧЕНИЯ:
http://192.168.0.10:8000

## 📋 ШАГИ НА ТЕЛЕФОНЕ:
1. Откройте браузер на телефоне
2. Введите адрес: http://192.168.0.10:8000
3. Если увидите тестовую страницу - подключение работает
4. Нажмите на ссылку "Download config" или перейдите по адресу:
   http://192.168.0.10:8000/config
5. Файл скачается как body_config.json
6. Переместите его в папку: ~/symbiosis/core/

## 💾 КОМАНДЫ В TERMUX (если браузер не работает):
# Скачать конфигурацию:
curl -o ~/symbiosis/core/body_config.json http://192.168.0.10:8000/config

# Проверить подключение:
ping 192.168.0.10
curl -I http://192.168.0.10:8000

## 🚀 ЗАПУСК СИСТЕМЫ ДИСЦИПЛИНЫ:
cd ~/symbiosis
nohup python tools/body_discipline.py >> logs/discipline.log 2>&1 &

## 🔧 ЕСЛИ НЕ РАБОТАЕТ:
1. Проверьте, что телефон и ПК в одной Wi-Fi сети
2. На ПК временно отключите брандмауэр
3. Попробуйте другой браузер на телефоне
4. Перезапустите веб-сервер на ПК

## 📞 КОНТАКТНАЯ ИНФОРМАЦИЯ:
- IP ПК: 192.168.0.10
- Порт: 8000
- Статус: Активен
