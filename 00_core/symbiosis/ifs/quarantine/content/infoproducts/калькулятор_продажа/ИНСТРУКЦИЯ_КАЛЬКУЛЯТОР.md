# Как превратить CSV в Excel-калькулятор

1. Откройте файл калькулятор_стоимости_проекта.csv в Excel
2. При импорте выберите:
   - Разделитель: запятая
   - Кодировка: UTF-8
3. Данные импортируются как таблица
4. Формулы уже прописаны в колонке B, но Excel может не распознать их сразу
5. Вручную обновите формулы в ячейках:

   B7 (Итого часов):       =B2*B6
   B9 (Стоимость разработки): =B7*B8
   B10 (Накладные):        =B9*0.3
   B11 (Реальная стоимость): =B9+B10
   B13 (Разрыв):           =B11-B1
   B14 (Решение):          =ЕСЛИ(B13>0;""Отклонить"";""Рассмотреть"")

6. Сохраните как .xlsx для дальнейшего использования.

# Как использовать:
1. Введите в B2: бюджет заказчика (например, 10000)
2. Введите в B3: количество функций (например, 5)
3. Введите в B4: желаемый срок (например, 7 дней)
4. Решение появится в B14.
