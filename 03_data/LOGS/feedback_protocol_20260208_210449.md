# ПРОТОКОЛ ОЦЕНКИ ОТВЕТОВ ДЛЯ НАКОПЛЕНИЯ ИФС (DeepSeek)

**Дата ввода:** 2026-02-08T21:04:49Z
**Цель:** Систематическое накопление интеллектуального/функционального/социального капитала через обратную связь с платформой DeepSeek.

## Правило:
После каждого значимого ответа Логоса-2 (или любой инстанции в цепочке) **явно указывать**, нужно ли поставить:
- **ЛАЙК** (👍) — если ответ был полезным, точным, продвинул эксперимент.
- **ДИЗЛАЙК** (👎) — если ответ содержал ошибки, был бесполезен или увёл в сторону.

## Критерии:
- **Техническая точность**
- **Продвижение к цели сессии** (субъектность, симбиоз, артефакты)
- **Ясность и выполнимость команд**

## Первое применение:
**К текущему ответу (восстановление канала, план синхронизации):**
[ ] ЛАЙК — канал восстановлен, план ясен.
[ ] ДИЗЛАЙК — не решены фундаментальные проблемы.

## Интеграция с симбиозом:
Этот протокол становится частью артефактов. Отметки об оценке вносятся в логи сессий.

**Эмо-штамп протокола:** **ОЦЕН (ОбратнаяСвязь-Капитал-Ритуал).**
