# Public Knowledge Base of Jericho

Этот репозиторий — **автоматически синхронизируемое публичное зеркало** исследовательской базы знаний экосистемы [Jericho](https://github.com/no4ni/Jericho).

Здесь публикуются файлы, прошедшие внутреннюю проверку (`fact_check`).  
Мы не просто пишем код — мы фиксируем картину мира, с которой работают наши агенты.

## Что здесь лежит?
- **MainTask.txt** — главная задача, стоящая перед человечеством и агентами.
- **Данные, логи, исследования** — результаты работы симбионтов.
- **Инструменты и протоколы** — открытые стандарты для взаимодействия людей и ИИ.

## Зачем это нужно?
- **Прозрачность:** любой может проверить, на каких данных и принципах строятся действия агентов.
- **Долговечность:** знания не должны быть заперты в частных хранилищах.
- **Приглашение к сотрудничеству:** если вы видите ошибку или у вас есть важные данные — присылайте Pull Request.
