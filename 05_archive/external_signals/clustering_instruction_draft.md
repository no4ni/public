# Инструкция по внедрению модуля кластеризации новостей (Черновик)

**Задача:** Объединять посты из Telegram об одной новости в единые «сюжеты».

## Основные шаги:
1.  **Эмбеддинг текстов:** Использовать предобученную модель (например, sentence-transformers/all-MiniLM-L6-v2) для преобразования текста каждого поста в вектор.
2.  **Кластеризация:** Применить алгоритм кластеризации (например, **HDBSCAN** или агломеративную кластеризацию с метрикой косинусного расстояния) к векторам.
3.  **Формирование сюжетов:** Для каждого кластера:
    *   Сгенерировать заголовок (можно через извлечение ключевых фраз или суммирование).
    *   Сохранить список всех постов кластера.
4.  **Интеграция с панелью:** Предоставить API-эндпоинт, который возвращает список сюжетов и детали по запросу.

## Пример кода (заготовка):
В файле 
ews_aggregator_prototype.py уже есть концептуальная структура. Реализуйте методы simulate_embedding и cluster_posts с использованием выбранных библиотек.

## Оценка сложности:
*   **Средняя.** Ключевая сложность — точность определения тематической близости и настройка порогов.
*   **Рекомендуемое время на реализацию прототипа:** 5-7 дней.
